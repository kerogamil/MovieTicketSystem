A desktop application for managing cinema ticket reservations, developed using **JavaFX** and **Java** following the **MVC architecture**.

---

## Features

### User
- User registration & login
- Browse available movies
- View movie details and showtimes
- Select seats
- Reserve tickets
- Payment simulation
- View reservation details

### Admin
- Admin login
- Add / update / remove movies
- Manage halls and shows
- View reservations
- Manage seats availability
